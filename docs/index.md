# OpenObs

OpenObs is an AI-native observability platform for incident investigation, dashboard generation, and alert lifecycle management.

## What OpenObs does

- Generates dashboards from natural-language prompts against real infrastructure metrics
- Investigates incidents by collecting evidence, evaluating hypotheses, and writing reports
- Creates and manages alert rules with conversational workflows
- Streams agent progress live so operators can see what the system is doing
