# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-04-12

### Added
- AI-driven dashboard generation from natural language
- Automated investigation of production incidents with LLM agents
- Prometheus/VictoriaMetrics integration with metric discovery
- Alert rule creation and management via conversational AI
- Multi-provider LLM gateway (Anthropic, OpenAI, Google)
- Role-based access control with JWT authentication
- Webhook system for inbound/outbound event integration
- Execution engine with guardrails and approval workflows
- Real-time streaming via SSE and WebSocket
- SQLite and PostgreSQL persistence backends
- Adapter SDK for building custom integrations
